# Estima Finance - Calculator Credit (Next.js)

## Instrucțiuni pentru rulare locală:

1. Instalează Node.js: https://nodejs.org
2. Descarcă și extrage acest fișier .zip
3. În terminal, rulează:

   cd calculator-credit-estima-finance
   npm install
   npm run dev

4. Accesează în browser: http://localhost:3000
